# Sudoku

Soduko and Sudoku solver app.

## Copyright

(C) 2014-2023 Denis Meyer

## Features

* Play Sudoku
* "Hint" a next possible step
* Check whether Sudoku is valid and solvable
* Solve entire Sudoku
* 4 difficulties
* Save and load games
